Personal Modern Poems,65 modern chinese pomes composed by myself which names '听说'&lt;listening> that expresses the thought of life, love and surrounding during my bachelor period, 个人本科期间创作的个人现代中文诗集&lt;听说>,一共65篇,分成七个章节.当时真的是风骚,多情的很....

# 项目介绍

&ensp;&ensp;诗词是人类文化艺术瑰宝，诗词作为一种表达人情感的简短表达，在我国的历史文化潮流中占据着重要角色，如我们所熟知的唐诗宋词，近代诗人海子，北岛等等。      
人类创作诗歌文学已经有很长的历史，但就最近而言,随着自然语言处理的发展,计算文学作为社会计算中的一个应用点一度成为热点，例如微软机器人微软小冰创作的<阳光失了玻璃窗>，清华大学孙茂松老师团队研制开发的"九歌"古诗词创作系统，两个分别从现代和古代诗歌文体两个方面进行了机器诗歌创作的实验，取得了不错的效果，也推动了包括seq2seq模型的诗歌创作的模型的发展。     
&ensp;&ensp;本人在五年前,还在读本科的时候，估计还是比较年轻，内心感情比较丰富，当时也一度比较悲观，零零散散地写了一些酸溜溜的文字，细细数来，写了7个主题一共65首诗，取名为<听说>。   
&ensp;&ensp;本项目将对其进行总结，对此有兴趣的朋友可以看一看，提一提意见。未来，我将尝试实验从机器创作的角度触出发，进行诗歌创作。
